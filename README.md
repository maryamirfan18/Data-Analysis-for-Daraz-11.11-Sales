# Data-Analysis-for-Daraz-11.11-Sales

## Project Overview

This project focuses on the development of a **centralized Data Warehouse** using the Daraz e-commerce dataset. The primary objective is to consolidate data from multiple sources, process and transform it into a structured format, and enable data-driven decision-making through comprehensive analysis and visualization.

The project implements a complete **ETL (Extract, Transform, Load)** process to extract raw data, perform data cleaning and transformation, and load the processed information into a centralized SQLite database for efficient storage, querying, and reporting.

### Key Objectives
- Build a centralized data repository
- Clean and transform raw datasets
- Perform data denormalization for analytical reporting
- Store structured data in a SQLite database
- Generate business insights through visualizations and dashboards
